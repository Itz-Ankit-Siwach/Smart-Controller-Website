Hand Gesture Controller

Revolutionize your system with touch-free control using advanced computer vision technology.

Our system leverages MediaPipe for real-time hand tracking and OpenCV for gesture recognition, enabling seamless control of presentations through intuitive hand movements. The intuitive interface built with Tkinter provides customizable controls for a personalized experience.

Key Features:
  • Slide navigation (next/previous) with simple finger gestures
  • Mouse pointer control and click actions
  • Pinch-to-zoom functionality
  • Volume control gestures
  • Voice dictation activation
  • Tab switching between applications

Complete Gesture Reference Guide

Voice Dictation

Middle Finger Extended

Activates Windows voice-to-text dictation for hands-free typing

How to perform:
  1. Extend middle finger straight up
  2. Keep other fingers folded down
  3. Hold for 1 second to activate
Tip: Works best in quiet environments

Next Slide

Pinky Finger Extended

Advances to the next slide in presentation mode

Works with:
PowerPoint Google Slides PDF Readers
Tip: Works during full-screen presentations

Previous Slide

Thumb Extended

Returns to the previous slide or goes back in navigation

Cooldown:
0.3 seconds between gestures

Left Click

Thumb Index Pinky Extended

Simulates a standard mouse left-click action

Pointer Control:

Use two-finger point gesture to move cursor before clicking

Tip: Hold gesture for 0.5s for drag operations

Right Click

Index Ring Pinky Extended

Simulates a standard mouse right-click action

Context Menu:

Works with all standard right-click context menus in Windows

Tip: 1-second cooldown prevents accidental activation

Media Controls

Multiple Gestures

Control volume, media playback, and navigation

Available Commands:
  • Index + Pinky: Volume Up
  • Middle + Pinky: Volume Down
  • Index + Middle: Arrow Up
  • Ring + Pinky: Arrow Down

Tab Switching

Full Hand Swipe Motion

Switch between browser/application tabs

Directions:
  • Swipe Right: Next Tab (Ctrl+Tab)
  • Swipe Left: Previous Tab (Ctrl+Shift+Tab)
Tip: Requires at least 30cm horizontal movement

Zoom Control

Two Hands Pinch Motion

Zoom in/out by moving hands closer/further apart

Supported Applications:
Browsers PDF Viewers Image Editors

Scroll Control

Two Fingers Vertical Motion

Scroll up/down by moving fingers vertically

Our system requires:

  • Windows 10/11
  • Webcam (720p minimum resolution)
  • 4GB RAM minimum (8GB recommended)
  • Python 3.8+ (included in installer)

You can customize gestures through the Control Panel:

  1. Open the application
  2. Click "Customize" button
  3. Adjust gesture sensitivity sliders
  4. Map gestures to different actions
  5. Save your configuration

If your webcam isn't detected, try these steps:

  • Ensure no other application is using the camera
  • Check your system permissions for camera access
  • Update your webcam drivers
  • Try a different USB port if using external camera
  • Restart the application

If issues persist, contact our support team with your camera model details.
